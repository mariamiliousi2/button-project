# Student Attention Tracker - React Native Expo App

Εφαρμογή παρακολούθησης προσοχής φοιτητών για Android/iOS.

## Χαρακτηριστικά

✅ **Εισαγωγή Student ID** - Εισάγετε το Student ID σας κατά την πρώτη εκκίνηση
✅ **Καταγραφή Προσοχής** - Καταγράψτε την προσοχή σας με δύο κουμπιά: "ΠΡΟΣΕΧΩ" και "ΒΑΡΙΕΜΑΙ"
✅ **Ιστορικό Logs** - Δείτε το ιστορικό των καταγραφών σας σε δύο λίστες
✅ **Εξαγωγή CSV** - Εξάγετε όλα τα δεδομένα σε αρχείο CSV

## Εγκατάσταση

### Προαπαιτούμενα
- Node.js (v16 ή νεότερη)
- npm ή yarn
- Expo CLI: `npm install -g expo-cli`

### Βήματα Εγκατάστασης

1. **Εγκατάσταση dependencies:**
   ```bash
   npm install
   ```

2. **Εκκίνηση της εφαρμογής:**
   ```bash
   npm start
   ```
   ή
   ```bash
   expo start
   ```

3. **Εκτέλεση σε συσκευή:**
   - **Android:** `npm run android` ή πατήστε `a` στο Expo CLI
   - **iOS:** `npm run ios` ή πατήστε `i` στο Expo CLI (μόνο σε macOS)
   - **Φυσική συσκευή:** Σκανάρετε το QR code με την εφαρμογή Expo Go

## Δομή Εφαρμογής

### Οθόνη 1: Student ID Input
- Εμφανίζεται μόνο κατά την πρώτη εκκίνηση
- TextInput για εισαγωγή Student ID
- Αποθήκευση στο AsyncStorage

### Οθόνη 2: Κύρια Οθόνη
- **Δύο κουμπιά δράσης:**
  - 🟢 Πράσινο κύκλο "ΠΡΟΣΕΧΩ" (150x150)
  - 🔴 Κόκκινο κύκλο "ΒΑΡΙΕΜΑΙ" (150x150)
- **Δύο λίστες logs:**
  - Λίστα ΠΡΟΣΕΧΩ (πράσινη)
  - Λίστα ΒΑΡΙΕΜΑΙ (κόκκινη)
- **Κουμπί εξαγωγής CSV** στο κάτω μέρος

## Λειτουργίες

### Καταγραφή
Κάθε φορά που πατάτε ένα κουμπί, δημιουργείται ένα log με:
- `timestamp`: Η ημερομηνία και ώρα (μορφή: "2026-02-23T12:49:00")
- `action`: "ΠΡΟΣΕΧΩ" ή "ΒΑΡΙΕΜΑΙ"

### Εξαγωγή CSV
Το αρχείο CSV περιέχει:
- `student_id`: Το Student ID σας
- `action`: Η ενέργεια (ΠΡΟΣΕΧΩ/ΒΑΡΙΕΜΑΙ)
- `timestamp`: Η χρονική σήμανση

**Σημαντικό:** Για να εξάγετε το CSV, θα ζητηθούν δικαιώματα πρόσβασης στο σύστημα αρχείων της συσκευής σας.

## Troubleshooting

### Πρόβλημα με permissions στο Android
Βεβαιωθείτε ότι έχετε δώσει δικαιώματα στην εφαρμογή από τις ρυθμίσεις της συσκευής.

### Δεν εμφανίζονται τα logs
Πατήστε τα κουμπιά "ΠΡΟΣΕΧΩ" ή "ΒΑΡΙΕΜΑΙ" για να δημιουργήσετε logs.

### Το CSV δεν εξάγεται
1. Βεβαιωθείτε ότι έχετε δώσει δικαιώματα
2. Ελέγξτε ότι υπάρχουν logs για εξαγωγή
3. Δοκιμάστε να επιλέξετε έναν διαφορετικό φάκελο
